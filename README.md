# ProjetoINSS

Bem vindo ao Projeto Cálculo Desconto INSS

O projeto do WebApi se encontra no caminho INSS/INSS.sln. Basta o abrir e executar localmente.

A chamada para o método é: /api/CalculadorInss/CalcularInss?pData={data}pSalario={salario}

Ex: http://localhost:9942/api/CalculadorInss/CalcularInss?pData=01/01/2011&pSalario=4000.00
